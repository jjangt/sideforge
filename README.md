# SideForge

> AI와 함께 만드는 나만의 브랜드와 두 번째 수입

## Overview

SideForge는 사용자의 관심사, 경험, 성향, 목표를 분석하여 수익화 가능한 브랜드를 발굴하고, 브랜드를 생성하며, 성장 방향을 제안하는 AI 공동창업자 플랫폼입니다.

## Quick Start

```bash
npm install
npm run web        # 웹 실행
npm run android    # Android 실행
npm run ios        # iOS 실행
```

## Tech Stack

- **Framework**: Expo (React Native) + Expo Router
- **Language**: TypeScript
- **Styling**: Nativewind (Tailwind CSS)
- **State**: Zustand
- **AI**: Provider Factory Pattern (Mock → Claude → OpenAI)
- **Backend**: Supabase (선택)
