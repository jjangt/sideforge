# Mock Response Specification

## 개요
데모용 Mock AI 응답 스펙입니다. 프로필 키워드 기반으로 시나리오를 분기합니다.

## 시나리오 분기 로직
- 음식/디저트/요리 키워드 → 시나리오 A (디저트 큐레이션)
- 테크/생산성/AI 키워드 → 시나리오 B (생산성 콘텐츠)
- 여행/라이프스타일 키워드 → 시나리오 C (여행 브랜드)
- 기본 → 시나리오 A

## 응답 지연
Mock Provider는 실제 AI 호출처럼 1~3초 지연을 시뮬레이션합니다.
