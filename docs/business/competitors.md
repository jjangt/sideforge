# 경쟁사 분석

## 직접 경쟁사

### 1. VidIQ (vidiq.com)
- **분야**: YouTube 분석 + SEO
- **가격**: 무료 / $16.58/월 / $39.17/월
- **강점**: YouTube 특화, 키워드 점수, 경쟁 트래킹
- **약점**: YouTube만, UI 복잡, AI 피드백 없음
- **우리의 차별점**: AI 기반 구체적 액션 제안, 더 심플한 UI

### 2. Social Blade (socialblade.com)
- **분야**: YouTube/Instagram/TikTok 통계
- **가격**: 무료 / $3.99/월 / $39.99/월
- **강점**: 무료 기본 통계, 등급 시스템, 인지도 높음
- **약점**: 디자인 구식, AI 없음, 숫자만 나열
- **우리의 차별점**: "왜" + "어떻게 고칠지" AI 피드백

### 3. Metricool (metricool.com)
- **분야**: SNS 분석 + 스케줄링
- **가격**: 무료(1계정) / €18/월 / €45/월
- **강점**: 올인원, 깔끔한 대시보드
- **약점**: 분석 깊이 없음, AI 없음
- **우리의 차별점**: 분석에 집중, AI가 개선점 제안

### 4. Not Just Analytics (notjustanalytics.com)
- **분야**: Instagram 분석
- **가격**: 무료 / €19/월 / €49/월
- **강점**: URL만 넣으면 즉시 분석, 진입장벽 0
- **약점**: Instagram만 지원
- **우리의 차별점**: 멀티 플랫폼 (YouTube + Blog + Instagram)

### 5. Iconosquare (iconosquare.com)
- **분야**: Instagram + TikTok 분석
- **가격**: €49/월 / €79/월 / €119/월
- **강점**: 리포트 디자인 좋음, 경쟁사 비교
- **약점**: 매우 비쌈, AI 없음
- **우리의 차별점**: 1/5 가격에 AI 분석 포함

## 경쟁사 공통 약점 = SideForge 기회

| 약점 | SideForge 대응 |
|------|---------------|
| 숫자만 보여줌 | AI가 원인 + 개선 방법 제안 |
| 플랫폼 1개만 지원 | YouTube + Blog + Instagram 통합 |

| 비쌈 | Plus ₩9,900 / Pro ₩29,900 |
