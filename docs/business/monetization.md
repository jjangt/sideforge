# Monetization

## 개요
SideForge 수익화 전략 문서입니다.

## 수익화 단계

### Phase 1 (MVP 이후)
- 무료 서비스로 사용자 확보
- 유료 전환 트리거 설계

### Phase 2
- Pro 구독 (AI 코치 무제한, 브랜드 복수 생성)
- 프리미엄 브랜드 템플릿

### Phase 3
- 도메인 연결 기능 과금
- 브랜드 성과 분석 리포트 과금
- 팀/에이전시 플랜

## KPI
- MAU (월간 활성 사용자)
- 브랜드 생성 수
- Free → Pro 전환율
- AI 코치 세션 수
