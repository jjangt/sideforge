# 개인정보처리방침

최종 수정일: 2025년 6월 22일

## 1. 수집하는 개인정보

| 구분 | 항목 | 수집 목적 |
|------|------|-----------|
| 필수 | 이메일 | 계정 식별, 로그인, 안내 발송 |
| 선택 | 이름 | 서비스 내 표시 |
| 자동 | IP, 접속 시간, 기기 정보 | 보안, 서비스 개선 |
| Google 로그인 시 | 이메일, 이름, 프로필 사진 | 계정 생성 및 표시 |

## 2. 개인정보 이용 목적

- 서비스 제공 및 계정 관리
- 결제 처리 및 구독 관리
- 서비스 개선 및 통계 분석
- 공지사항 및 마케팅 정보 발송 (동의 시)

## 3. 개인정보 보관 기간

| 상황 | 보관 기간 |
|------|-----------|
| 서비스 이용 중 | 회원 탈퇴 시까지 |
| 회원 탈퇴 | 즉시 삭제 (법령 의무 제외) |
| 구독 해지 (탈퇴 X) | 6개월 보관 후 삭제 |
| 결제 기록 | 5년 (전자상거래법) |

## 4. 개인정보 제3자 제공

원칙적으로 제3자에게 제공하지 않습니다. 예외:
- 이용자 동의 시
- 법령에 의한 요구 시

## 5. 개인정보 처리 위탁

| 수탁자 | 위탁 업무 |
|--------|-----------|
| Cloudflare | 서버 호스팅, 데이터 저장 |
| Stripe / 토스페이먼츠 | 결제 처리 |
| Google | OAuth 인증 |

## 6. 이용자의 권리

- 개인정보 조회, 수정, 삭제 요청 가능
- 마케팅 수신 동의 철회 가능
- 계정 삭제 요청 시 30일 내 처리

## 7. 보안 조치

- 비밀번호 단방향 암호화 (SHA-256 + salt)
- HTTPS 통신 암호화
- JWT 토큰 기반 인증 (7일 만료)
- 환경변수로 민감 정보 관리 (코드에 키 미노출)

## 8. 쿠키 사용

- 로그인 세션 유지를 위해 토큰을 로컬 스토리지에 저장합니다.
- 광고 쿠키는 Free 플랜 이용자에게만 적용됩니다 (AdSense/카카오 애드핏).
