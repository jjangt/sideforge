# Brand Preview

## 개요
생성된 브랜드를 실제 공개 페이지처럼 시각화하여 "내 브랜드가 생겼다"는 경험을 제공합니다.

## 구성 요소
- 로고 컨셉 영역
- 브랜드명 + 슬로건
- 브랜드 스토리
- 타겟 고객 소개
- 컬러 팔레트 시각화
- 첫 콘텐츠 미리보기
- SNS 프로필
- Powered by SideForge

## 경로
`/brand/[id]/preview`
