# AI Coach

## 개요
단순 챗봇이 아닌, 브랜드 컨텍스트를 인식하는 AI 공동창업자 대화 인터페이스입니다.

## 핵심 차별점
- 현재 브랜드 상태 인식
- 30일 플랜 진행 상황 반영
- 구체적 액션 아이템 제시
- 추천 질문 제공

## 추천 질문 예시
- "이번 주에 뭐부터 하면 좋을까?"
- "이 브랜드로 돈을 벌려면 뭐가 먼저 필요할까?"
- "내 브랜드가 너무 흔하지 않아?"
- "첫 콘텐츠는 뭘 올리면 좋을까?"
- "이 브랜드를 30일 동안 키우려면 어떻게 해야 할까?"
