# YouTube 분석 v2 — "다음 영상 전략" 킬링 포인트

## 문제 인식

현재 분석 결과가 제공하는 정보(조회수 높다/낮다, 업로드 자주 해라)는  
YouTube Studio에서 직접 확인 가능한 수준이라 **유료 결제할 동기가 부족**.

## 핵심 방향 전환

> **"다음에 뭘 올려야 터지는지"를 데이터로 알려주는 서비스**

유튜버가 돈 내고 살 만한 정보 = 스스로는 분석하기 어려운 것:
1. 내 채널에서 "터진 영상"의 공통 패턴이 뭔지
2. 그 패턴을 기반으로 다음에 뭘 만들면 되는지
3. 내 영상이 안 되는 구체적 이유 (길이? 제목? 타이밍?)

## 데이터 기반 패턴 분석 (코드에서 계산)

### 현재 수집 중인 데이터
- 영상별: 제목, 조회수, 좋아요, 댓글 수, **길이(초)**, 게시일
- 채널: 구독자, 총 조회수, 영상 수
- 댓글: 상위 3개 영상의 시청자 댓글 (채널주인 제외)

### `preAnalyze`에서 계산할 패턴들

#### 1. 바이럴 공식 분석
```
- 조회수 상위 30% 영상 vs 하위 30% 영상 비교
- 비교 항목:
  - 평균 영상 길이 (초)
  - 평균 제목 길이 (글자수)
  - 업로드 요일 분포
  - 제목에 포함된 공통 키워드
```

**출력 예시:**
> "조회수 상위 영상의 공통점: 평균 길이 8분(하위는 22분), 제목에 'vs' 'TOP' 포함 비율 80%, 주로 화~목 업로드"

#### 2. 영상 길이 vs 성과 상관관계
```
- 구간별 평균 조회수:
  - 0~3분 (Shorts급)
  - 3~8분
  - 8~15분
  - 15분+
- 가장 성과 좋은 구간 식별
```

**출력 예시:**
> "이 채널은 8~15분 영상이 평균 조회수 5만으로 최적. 15분+ 영상은 평균 1.2만으로 73% 하락"

#### 3. 업로드 타이밍 분석
```
- 요일별 평균 조회수
- 가장 성과 좋은 요일 vs 나쁜 요일
```

**출력 예시:**
> "화요일 업로드 영상 평균 조회수 8만 vs 일요일 2만. 화~목 업로드가 최적"

#### 4. 제목 패턴 분석
```
- 조회수 상위 영상 제목에서 빈출 키워드 추출
- 제목 길이 vs 조회수 상관관계
- 숫자/특수문자 포함 여부 vs 성과
```

**출력 예시:**
> "제목에 숫자 포함 시 평균 조회수 2.3배. 제목 20자 이하가 30자+ 대비 1.8배 성과"

#### 5. 시청자 요청 추출 (댓글 기반)
```
- 댓글에서 "다음에", "해주세요", "궁금", "요청" 등 키워드 포함 댓글 식별
- AI에게 "시청자가 요청하는 콘텐츠 목록" 추출 요청
```

## 리포트 구조 변경

### Before (현재)
```
점수 → 강점 → 단점 → 개선액션 → 추천콘텐츠
```

### After (개선)
```
점수 → 최근 영상 성과(+댓글) → 바이럴 공식 → 단점 진단 → 다음 영상 전략 → 추천콘텐츠
```

### 새로운 섹션: "🔥 바이럴 공식" (킬링 포인트)
```
이 채널에서 터진 영상의 공통점:
- 최적 영상 길이: 8~12분
- 최적 업로드 타이밍: 화~목
- 제목 패턴: 숫자 + 비교 형식 (예: "TOP 5", "A vs B")
- 주제: [데이터에서 추출한 고성과 주제]

다음 영상 추천:
"[구체적 제목 예시]" — 길이 10분 내외, 수요일 오후 업로드 권장
근거: 유사 패턴 영상 평균 조회수 X만회
```

## 기술 구현 계획

### Phase 1: preAnalyze 확장 (코드)
파일: `api/src/prompts/youtube.ts`의 `preAnalyze` 함수

```typescript
// 추가할 분석 항목
interface PatternAnalysis {
  optimalDuration: { min: number; max: number; avgViews: number };
  bestDay: { day: string; avgViews: number };
  worstDay: { day: string; avgViews: number };
  titlePatterns: { keyword: string; avgViews: number }[];
  topVsBottom: {
    topAvgDuration: number;
    bottomAvgDuration: number;
    topAvgTitleLength: number;
    bottomAvgTitleLength: number;
  };
}
```

### Phase 2: 프롬프트 재구성
- AI에게 패턴 분석 결과를 넘기고
- "다음 영상 전략"을 구체적으로 작성하게 함
- 영상 제목 예시 + 길이 + 타이밍까지 포함

### Phase 3: 프론트 UI
- "🔥 바이럴 공식" 섹션 추가
- 최적 길이/요일/제목 패턴을 시각적으로 표시

## 경쟁 우위

- vidIQ/TubeBuddy: 키워드 도구 중심. 채널 전체 패턴 분석은 약함.
- 우리: **"네 채널의 바이럴 공식"을 데이터로 추출** → 이건 다른 도구에서 안 해줌.
- 무료 AI로도 가능한 이유: 패턴 분석은 코드에서 하고, AI는 문장만 작성.
