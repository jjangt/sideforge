# YouTube 분석 기능

## 개요

YouTube 채널 URL을 입력하면 AI가 채널을 분석하고 구체적인 개선 액션을 제공합니다.

## 사용자 플로우

```
1. URL 입력 (youtube.com/@채널명 또는 youtube.com/channel/ID)
2. 채널 확인 (이 채널이 맞나요?)
3. 분석 중 (로딩, 15~30초)
4. 리포트 표시
```

## 수집 데이터 (YouTube Data API v3)

### 채널 정보
- 채널명, 프로필 이미지
- 구독자 수
- 총 영상 수
- 총 조회수
- 채널 개설일
- 채널 설명

### 최근 영상 (20~50개)
- 제목
- 썸네일
- 조회수, 좋아요, 댓글 수
- 게시일
- 영상 길이
- 태그

### 댓글 (Pro 플랜)
- 최근 영상 상위 댓글 50개씩
- 댓글 좋아요 수
- 답글 수

## AI 분석 항목

### 기본 리포트 (Free)

| 항목 | 설명 |
|------|------|
| 채널 점수 | 100점 만점 종합 점수 |
| 성장성 | 구독자 대비 최근 조회수 추세 |
| 한줄 요약 | "콘텐츠 퀄리티는 높으나 업로드 빈도가 낮습니다" |

### 상세 리포트 (Plus)

| 항목 | 설명 |
|------|------|
| 콘텐츠 분석 | 영상별 성과 비교 + 잘된/못된 영상 이유 |
| 제목 분석 | 클릭률 높은 제목 패턴 vs 현재 제목 |
| 업로드 패턴 | 요일/시간대별 성과 분석 |
| 추천 콘텐츠 | "이런 영상을 만들어보세요" 3개 제안 |
| 개선 액션 | 지금 바로 할 수 있는 것 3가지 |

### 프로 리포트 (Pro)

| 항목 | 설명 |
|------|------|
| 댓글 감성 분석 | 긍정/부정/요청 분류 |
| 썸네일 분석 | 클릭률 높은 썸네일 특징 비교 |
| 경쟁 채널 비교 | 같은 카테고리 인기 채널과 비교 |
| 이탈 원인 분석 | 조회수 대비 좋아요 비율 낮은 영상 원인 |
| 수익화 제안 | 협찬, 멤버십, 상품 등 수익화 전략 |

## AI 프롬프트 구조

```
[시스템] 당신은 YouTube 채널 성장 전문 컨설턴트입니다.
데이터 기반으로 분석하고, 구체적이고 실행 가능한 액션을 제안합니다.

[데이터]
채널명: {name}
구독자: {subscribers}
최근 30일 조회수: {recentViews}
영상 목록: [{title, views, likes, comments, publishedAt}, ...]

[질문]
1. 이 채널의 강점과 약점을 분석해주세요.
2. 가장 성과가 좋은 영상의 공통점은 무엇인가요?
3. 조회수를 높이기 위해 지금 바로 할 수 있는 3가지를 제안해주세요.
4. 다음에 만들면 좋을 영상 주제 3개를 추천해주세요.
```

## API 엔드포인트 (Cloudflare Workers)

```
POST /api/analyze/youtube
Body: { url: "https://youtube.com/@channelName" }
Response: { reportId: "abc123", status: "processing" }

GET /api/report/{reportId}
Response: { score, summary, details, actions, ... }
```

## YouTube Data API 쿼터

| 작업 | 비용 (쿼터) | 1회 분석 사용량 |
|------|------------|----------------|
| 채널 정보 | 1 | 1 |
| 영상 목록 | 1 | 1 |
| 영상 상세 (50개) | 1 | 1 |
| 댓글 (50개/영상) | 1 | 5~10 |
| **합계** | - | **~13 쿼터** |

일일 쿼터 10,000 ÷ 13 = **약 770회 분석/일 가능**
