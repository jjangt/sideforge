# Requirements

## 개요

SideForge MVP 기능 요구사항 정의입니다.

## 필수 기능 (MVP)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 프로필 입력 | 관심사, 강점, 목표 등 8개 필드 | 1순위 |
| 브랜드 추천 | AI 기반 3개 방향 추천 | 1순위 |
| 브랜드 생성 | 12개 브랜드 요소 생성 | 1순위 |
| 브랜드 미리보기 | 공개 페이지 형태 미리보기 | 1순위 |
| 대시보드 | 점수, 오늘 액션, 진행률 | 1순위 |
| AI 코치 | 공동창업자형 대화 | 2순위 |
| 30일 플랜 | 실행 계획 상세 뷰 | 2순위 |

## 비기능 요구사항

- Web / iOS / Android 지원
- 로컬 상태 우선 (오프라인 동작)
- AI Provider 교체 가능 구조
- Guardrails 적용 (AI 안전성)

## 제외 (MVP)

- 실제 SNS 연동
- 결제 기능
- 사용자 간 공유
