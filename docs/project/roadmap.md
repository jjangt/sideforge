# Roadmap

## Phase 1 — MVP (YouTube 분석)

### 인프라
- [ ] Cloudflare Workers 백엔드 세팅
- [ ] Cloudflare D1 (DB) 세팅
- [ ] Cloudflare Workers AI 연동 (Llama 3.1)
- [ ] YouTube Data API v3 연동

### 인증
- [ ] 이메일 + 비밀번호 로그인
- [ ] Google OAuth 로그인
- [ ] 세션/토큰 관리
- [ ] 계정별 분석 횟수 추적

### 핵심 기능
- [ ] YouTube 채널 URL 입력
- [ ] 채널 기본 정보 수집 (구독자, 영상 수, 총 조회수)
- [ ] 최근 영상 목록 + 개별 통계 수집
- [ ] AI 분석 리포트 생성 (점수 + 피드백 + 개선 액션)
- [ ] 분석 결과 저장 + 계정 내 조회

### 프론트엔드
- [ ] 랜딩 페이지 리디자인 (분석 서비스 소개)
- [ ] 로그인/회원가입 페이지
- [ ] URL 입력 페이지
- [ ] 분석 로딩 화면
- [ ] 리포트 페이지
- [ ] 마이페이지 (분석 히스토리)

### 무료 제한
- [ ] 계정당 3회 분석 (평생)
- [ ] 기본 리포트만 제공 (점수 + 요약)
