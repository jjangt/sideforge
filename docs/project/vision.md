# Vision

## 개요

SideForge는 크리에이터와 브랜드 운영자의 콘텐츠를 AI로 분석하고, 구체적인 개선 액션을 제공하는 플랫폼입니다.

## 핵심 가치

> "숫자가 아니라 액션을 준다"

기존 분석 도구: 조회수 1,234 | 좋아요 56 | 구독자 +12
SideForge: "이 영상은 썸네일 클릭률이 낮습니다. 비슷한 주제 인기 채널들은 [질문형 제목 + 얼굴 클로즈업]을 사용합니다. 수정 제안: ..."

## 문제 정의

1. 크리에이터들은 콘텐츠를 올리지만, "왜 안 되는지" 모른다
2. 기존 분석 도구는 숫자만 보여주고, 개선 방법을 알려주지 않는다
3. 전문 컨설팅은 비싸고 (월 수십~수백만원), 소규모 크리에이터는 접근 불가

## 해결책

- URL 입력만으로 즉시 AI 분석
- 구체적 개선 액션 제안 (무엇을, 왜, 어떻게)
- 잘 되는 채널과의 비교 벤치마킹
- 플랫폼 통합 (YouTube + Blog + Instagram)

## 타겟 사용자

| 단계 | 사용자 | 니즈 |
|------|--------|------|
| 초기 | 구독자 100~10,000 유튜버 | 성장 방법을 모름 |
| 중기 | 블로그/인스타 운영자 | 협찬/수익화 하고 싶음 |
| 확장 | 소규모 마케팅 팀 | 여러 채널 동시 관리 |

## 경쟁 우위

| 경쟁사 | 약점 | SideForge |
|--------|------|-----------|
| Social Blade | 숫자만 나열 | AI가 원인 분석 + 액션 제안 |
| VidIQ | YouTube만, 복잡 | 멀티 플랫폼, 심플 |
| Metricool | AI 피드백 없음 | AI 기반 개선 제안 |
| Not Just Analytics | Instagram만 | 통합 분석 |

## 기술 전략

| 항목 | 선택 | 이유 |
|------|------|------|
| 프론트엔드 | Expo (React Native Web) | 웹+앱 동시 지원 |
| 백엔드 | Cloudflare Workers | 무료 서버리스 |
| AI | Cloudflare Workers AI (Llama 3.1) | 무료 티어, 서버리스 |
| DB | Cloudflare D1 (SQLite) | 무료, Workers 통합 |
| 데이터 수집 | YouTube Data API, 네이버 API | 무료, 합법 |
| 배포 | Cloudflare Pages | 무료 |

## 수익 모델

URL 입력 → 무료 프리뷰(기본 점수) → 상세 분석은 유료 구독

## 장기 비전

1. Phase 1: YouTube 분석 MVP
2. Phase 2: Blog + Instagram 추가, Plus 플랜
3. Phase 3: Pro 플랜, 경쟁사 비교, 정기 모니터링
4. Phase 4: AI Agent (자동 콘텐츠 제안, 자동 게시)
