# Privacy

## 개요

SideForge 개인정보 처리 방침입니다.

## 수집 정보 (MVP)

- 관심사, 좋아하는 것, 잘하는 것 (텍스트)
- 성격, 목표, 원하는 분위기 (텍스트)
- 목표 수익, 주당 시간 (숫자)

## 수집하지 않는 정보

- 실명
- 이메일 (Supabase Auth 연동 전)
- 전화번호
- 주소
- 결제 정보

## 데이터 저장

- MVP: 로컬 기기(AsyncStorage)에만 저장
- 향후: Supabase (암호화 + RLS)

## 삭제 정책

- 로컬 데이터: 앱 삭제 시 자동 삭제
- 서버 데이터 (향후): 사용자 요청 시 30일 내 삭제
